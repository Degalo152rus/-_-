# Подключение иконок Font Awesome

## Для использования иконки калькулятора в форме

Добавьте в `<head>` вашей страницы перед другими стилями:

```html
<!-- Font Awesome для иконок -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.2/css/all.min.css">
```

Или используйте официальный CDN:

```html
<link rel="stylesheet" href="https://use.fontawesome.com/releases/v6.4.2/css/all.css">
```

## Альтернативные варианты иконок

Если хотите использовать другую иконку вместо калькулятора, замените класс в HTML:

```html
<!-- Калькулятор (текущая) -->
<i class="fas fa-calculator"></i>

<!-- Другие варианты: -->
<i class="fas fa-truck"></i>           <!-- Грузовик -->
<i class="fas fa-shipping-fast"></i>   <!-- Быстрая доставка -->
<i class="fas fa-chart-line"></i>      <!-- График -->
<i class="fas fa-route"></i>           <!-- Маршрут -->
<i class="fas fa-money-bill-wave"></i> <!-- Деньги -->
<i class="fas fa-percentage"></i>      <!-- Процент -->
<i class="fas fa-coins"></i>           <!-- Монеты -->
```

Посмотреть все доступные иконки: https://fontawesome.com/icons
