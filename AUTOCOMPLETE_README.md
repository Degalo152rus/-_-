# Автокомплит для формы калькулятора

## Установка

Добавьте следующие файлы в ваш HTML перед закрывающим тегом `</body>`:

```html
<!-- CSS для автокомплита -->
<link rel="stylesheet" href="css/autocomplete.css">

<!-- JavaScript для автокомплита -->
<script src="js/autocomplete.js"></script>
```

## Функциональность

### Основные возможности:
- ✅ Автоматическое подсказывание городов при вводе
- ✅ Фильтрация списка городов по введенному тексту
- ✅ Показ всех доступных городов при клике на пустое поле
- ✅ Навигация по списку с помощью клавиш ↑ и ↓
- ✅ Выбор города клавишей Enter
- ✅ Плавная анимация появления списка
- ✅ Адаптивный дизайн для мобильных устройств

### Города в базе данных:
Скрипт содержит 70+ крупнейших городов России, включая:
- Москва, Санкт-Петербург, Новосибирск
- Екатеринбург, Казань, Нижний Новгород
- И многие другие...

## Настройка

### Добавление новых городов:

Откройте файл `js/autocomplete.js` и добавьте города в массив `cities`:

```javascript
const cities = [
    'Москва',
    'Санкт-Петербург',
    'Ваш новый город',
    // ... остальные города
];
```

### Изменение стилей:

Откройте файл `css/autocomplete.css` и измените нужные параметры:

```css
.autocomplete-list {
    max-height: 250px;  /* Высота списка */
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.15); /* Тень */
}

.autocomplete-item {
    padding: 12px 15px; /* Отступы */
    font-size: 14px;    /* Размер шрифта */
}
```

### Изменение количества показываемых результатов:

В файле `js/autocomplete.js` найдите строку:

```javascript
).slice(0, 10); // Показываем максимум 10 результатов
```

Измените число `10` на нужное вам количество.

## Использование

Просто начните вводить название города в поля "Откуда" или "Куда":
1. При начале ввода появится список подходящих городов
2. Используйте мышь для выбора или клавиши ↑↓ для навигации
3. Нажмите Enter или кликните мышью для выбора города

## Совместимость

- ✅ Все современные браузеры (Chrome, Firefox, Safari, Edge)
- ✅ IE 11+ (с полифиллами)
- ✅ Мобильные браузеры (iOS Safari, Chrome Mobile)

## Техническая поддержка

При возникновении проблем проверьте:
1. Правильность путей к CSS и JS файлам
2. Наличие элементов с id `form_value-from` и `form_value-to`
3. Консоль браузера на наличие ошибок (F12 → Console)
